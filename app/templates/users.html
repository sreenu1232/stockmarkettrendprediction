About This Project

This Stock Market Prediction System is a Final Year Project that leverages machine learning (Linear Regression) and real-time financial data from Yahoo Finance to forecast future stock prices. The system provides interactive candlestick charts, historical trend analysis, and up to 365-day price predictions for any publicly listed stock ticker.
